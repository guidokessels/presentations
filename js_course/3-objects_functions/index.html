Type: undefined

A variable has the type undefined when it is declared, but no value has been assigned to it. 
var novalue;

alert(novalue);     // undefined

Checking the type with 'typeof'

Use the typeof statement to get the type of a variable or value
The result is a string with the type name 
typeof "This is a string"       // "string"
typeof 20                       // "number"
typeof { foo : 'bar' }          // "object"
typeof true                     // "boolean"
typeof function() {}            // "function"
typeof var_which_doesnt_exist   // "undefined"
typeof [1, 4, "Hello"]          // "object" ?!

Objects

What are objects?

  • An object is a key property storage container
  • Each property is labeled with a string which is called a 'key'
  • The key is used to set or get properties
  • Properties can be of any type
var arr = ['John', 26, 'Male'];
var obj = {
    'name'   : 'John',
    'age'    : 26,
    'gender' : 'Male'
};

Other ways to use objects

// Using .
var obj = {};
obj.name   = 'John';
obj.age    = 26;
obj.gender = 'Male';
alert(obj.name);        // 'John'

// Using brackets
var obj = {};
obj['name']   = 'John';
obj['age']    = 26;
obj['gender'] = 'Male';
alert(obj['name']);     // 'John'

Difference between objects and arrays

Objects have no length property

Objects don't have the push, pop, unshift and shift methods

You cannot use regular for loops

Looping over objects

Use a for ... in loop 
var obj = { name : 'John', age : 26, gender : 'Male' };

for (var i in obj) {
    // i is first 'name', then 'age', then 'gender'
    alert( "The value of '" + i + "' is " + obj[i] );
}

Advanced loops

Use the continue or break statements to end a loop iteration early

Works for both arrays and objects!

Continue

Use continue to end the current iteration and continue with the next iteration 
// Skip all odd numbers, and alert even numbers
for(var i = 1; i <= 10; i++) {
    if (i % 2 === 1) {
        continue;
    }

    alert(i);
}

Traversing through an array (HOW TO DO IT:)

for (var i = 0; i < fruits.length; i = i + 1) {
    alert(fruits[i]);
