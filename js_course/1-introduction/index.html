Values

There are 6 basic types of values in JavaScript
  • Numbers
  • Strings
  • Booleans
  • Objects
  • Functions
  • Undefined

Numbers

  • Numbers contain numeric values
  • Can be negative or positive
  • Fractional numbers are written by using a dot
  • You can use basic math operators
5 * 2       // 10
10 / 2.5    // 4
2 + 2       // 4
5 - 3 * 10  // -25
(5-3) * 10  // 20
100 % 3     // 1

Strings

  • Used to represent text
  • Anything between quotes is a String
  • You can use single and double quotes, as long as you're consistent
  • Use the + operator to concatenate strings
"John Doe"
'John Doe'
'Spil' + "Games"
'We can \'escape\' quotes'

Booleans

Boolean values are true and false 
2 > 1       // true
2 > 10      // false
3 < 2       // false

Using variables

Variables are declared by using the var statement

Variables are case-sensitive! 
var myname = "John Doe";
var four = 2 + 2;
var company = "Spil" + "Games";
var fortytwo = 10 * 4 + 10 - 8;

var forty = fortytwo - 2;
var forty = Fortytwo - 2; // ???

Comparison operators

Operators return a boolean result (true or false)
Used to determine equality or difference between variables or values
  • == Equal to
  • === Strict equal to
  • != Not equal to
  • !== Strict not equal to
  • < Less than
  • >= Less than or equal to
  • > Greater than
  • >= Greater than or equal to

Comparison operator examples

5 == '5'        
5 === '5'       
5 < 4           
12 >= 2         
4 === 10 / 2.5  
8 != 2 * 4;     
'John' === 'Jo' + 'hn' 

var ten = 2 * 5;
ten > 3
ten > 20

Logical operators

Operators return a boolean result (true or false)
Used to determine the logical difference between variables or values
  • && And
  • || Or
  • ! Not (flips the value given to it)

Logical operator examples

true && true        
true && false       
false && true       
false || true       
!false              
!true               

2 > 1 && 3 < 4      
3 < 4 && 5 > 6      

1 == 2 || 2 > 1         
2 < 3  || 3 == 1 + 2

Built-in functions

  • alert( message ) to display a message
  • prompt( text, value ) to ask for input
  • confirm( message ) to ask for confirmation

How to write JavaScript

Directly in your HTML

<script>
    var name = "John";
    alert( name );
</script>

In a .js file

Create a JavaScript file (eg, my_code.js) 
var name = "John";
alert( name );
Include that JavaScript file in your HTML 
<script src="my_code.js"></script>

In your browser's console

Press F12 to open your browser's console
Write your JavaScript code directly in the console
Works best in Chrome, or Firefox with Firebug installed
