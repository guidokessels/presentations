Arrays

What is an array

An array is a list of things

  • You can store any variable
  • You can store other arrays
  • Everything is stored in numbered 'drawers'
  • The drawer is called an index
  • The content of the drawer is called value

Using an Array (I)

Defining an array

you can create an array as a variable, initializing it by specifying square brackets.

The length property

you can ask JavaScript for the length of the entire array using the length property.

Retrieving elements

If you open a drawer with a certain number, you’ll get the element inside the drawer.

Example

var fruits = ["Lemon", "Orange", "Apple"];
fruits.length; //3
fruits[2]; //"Apple"

Using an Array (II)

Adding items (to the end)

you can add additional elements to an array with push.
fruits.push("Kiwi","Banana");

Removing items (from the end)

you can remove and retrieve the last element ofan array with pop.
var lastElement = fruits.pop(); // ["Lemon", "Orange", "Apple","Kiwi"]
lastElement; // "Banana"

Using an Array (III)

Adding items (to the beginning)

you can add additional elements to an array with unshift.
fruits.unshift("Melon"); //["Melon", "Lemon", "Orange", "Apple","Kiwi"]

Removing items (from the beginning)

you can remove and retrieve the first element ofan array with pop.
var firstElement = fruits.shift(); //["Lemon", "Orange", "Apple","Kiwi"]
firstElement; //"Melon"

Using an Array (IV)

Checking if an element is present in the array

fruits.indexOf("Kiwi"); //4
fruits.indexOf("apple"); //???

Convert arrays into strings and back

var citrusString = citrus.join(", "); //"Lemon, Orange"
var citrusArray  = citrusString.split(", "); //["Lemon", "Orange"]

Using an Array (V)

Traversing through an array (HOW NOT TO DO IT:)

alert(fruits[0]);
alert(fruits[1]);
alert(fruits[2]);
alert(fruits[3]);
alert(fruits[4]);

Using an Array (VI)

Traversing through an array (HOW TO DO IT:)

for (var i = 0; i < fruits.length; i = i + 1) {
    alert(fruits[i]);
}
