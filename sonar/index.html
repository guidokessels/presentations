How

create sonar-project.properties in the root of the codebase 

# required metadata
sonar.projectKey=spil:teens
sonar.projectName=Teens - PHP
sonar.projectVersion=1.0
sonar.language=php
 
# path to source directories (required)
sonar.sources=INC/app/gpconcept
